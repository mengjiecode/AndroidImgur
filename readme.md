# Android Imgur

Use Imgur API to allow user to view most viral images of the day/week/month, sorted by Newest/most popular.

(Currently only display first page of the content.)
- Link to Imgur API: https://api.imgur.com/

## Technologies:
- Retrofit2
- RxAndroid

## Language
- Java
